# 知识图谱图片链接分析

## 当前使用的图片系统

知识图谱当前使用的是字节跳动的AI图像生成API：

```
https://neeko-copilot.bytedance.net/api/text2image?prompt=[图片描述]&image_size=square_hd
```

这种方式的优点：
- 可通过文本描述动态生成相关图片
- 可以控制图片风格和质量
- 无需手动寻找和上传图片
- 与单词定义和主题高度匹配

## 替代图片资源选项

如果需要使用实际图片而非AI生成图片，可以考虑以下免费图片资源：

### 1. Unsplash (https://unsplash.com/)
- 高质量免费图片
- 可商用
- 搜索关键词：[单词标签] + "simple" + "isolated"

### 2. Pexels (https://www.pexels.com/)
- 免费图片和视频
- 可商用
- 搜索关键词：[单词标签] + "for kids" + "education"

### 3. Pixabay (https://pixabay.com/)
- 免费图片和视频
- 可商用
- 搜索关键词：[单词标签] + "cartoon" + "friendly"

## 建议

当前的AI生成图片系统已经很好地满足了知识图谱的需求，具有以下优势：
- 与单词定义高度相关
- 风格统一
- 无需额外维护
- 可以随时根据需求调整

如果需要替换图片，可以考虑以上资源，并保持链接格式的一致性。
